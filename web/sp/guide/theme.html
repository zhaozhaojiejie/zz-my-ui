# 内置主题

spWeb 4.x 的样式是用SCSS预处理，内置了一套变量。原则上在开发过程只允许使用内置的变量，不建议另起变量名，便于实现主题在线切换功能。 如需定制主题，只需重新给变量赋值即可。
