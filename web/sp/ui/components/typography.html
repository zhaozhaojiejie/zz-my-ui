# Typography 文本排版

文本的基本格式。 Typography 包含了4个子组件，分别是：

  • spTypography 用于显示文章容器
  • spTitle 用于显示标题
  • spParagraph 用于显示段落
  • spText 用于显示内联文本

# 何时使用

  • 当需要展示标题、段落、列表内容时使用，如文章/博客/日志的文本样式。
  • 当需要一列基于文本的基础操作时，如拷贝/省略/可编辑。
