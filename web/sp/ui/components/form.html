# Form 表单

spFrom 是组件库的重要组成部分，

它是一个表单项的容器，用来收纳表单项组件，提供排版、验证和数据处理功能。

# 代码示例

# 基础用法

spFrom继承 el-form的全部参数和事件。用法与el-from有差别， el-from需要与el-form-item结合使用。 spForm不需要显式调用，表单项组件内置了el-form-item， 不需要显式调用。

# 初始化数据

通过设置 model 属性回填表单数据

# 布局排版

与排版和外观相关的参数包括： itemWidth、footerAlign、inline、 labelWidth、labelPosition、size、 submitText、submittingText、resetText、footerBlock、 footerExpandBlock、footerFloat

# 验证

可以在spForm或表单项组件配置rules实现验证，用法与el-form、el-form-item一样

# 可折叠

设置 collapsible 可以实现折叠表单项，常用在筛选条件多的时候，把次要的折叠起来，节省空间。
