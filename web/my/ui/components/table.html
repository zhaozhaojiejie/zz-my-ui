# 开启列筛选

设置 column-filter 开启列筛选。 仅对普通列（非type的特殊类型及设置了prop的列）进行筛选 columns的项对象display默认为true，支持以下值：

  • true 默认值，初始化显示
  • false 初始不显示
  • always 一直显示，筛选器禁用
